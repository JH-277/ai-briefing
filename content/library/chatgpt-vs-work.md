# ChatGPT랑 '사내 AI'는 다른 거야?

## 첫 3초

집에서 쓰는 **ChatGPT**랑, 회사가 말하는 **“AI 도입”** — **같은 거 아닙니다.**

## 이런 적 있지 않나

개인적으로 ChatGPT 잘 써서  
“우리도 **AI 쓰자**, ChatGPT **팀 계정**이면 되지 않나?”  
IT팀이 **보안**, **법무**가 **개인정보** 얘기 꺼내는 장면.

## 쉽게 말하면

**개인 AI** = 내 **공책**에 메모 (편하지만 **회사 기밀** 올리면 위험)  
**사내 AI** = **회사 도서관 + 출입증** — **누가 뭘 넣었는지**, **로그**, **규정**이 있음  

같은 “AI”라도 **어디에 데이터가 가는지**, **누가 책임지는지**가 다릅니다.

## 회사에서 이렇게 쓰인다

| 상황 | 개인 ChatGPT | 사내 AI |
|------|-------------|---------|
| 이메일 초안 | 혼자 쓸 때 OK | **고객명·단가** 넣으면 위험 |
| 회의록 요약 | 빠름 | **승인된 도구** + **익명화** 후 |
| 규정 Q&A | 환각 가능 | **RAG + 사내 문서** |

**Before:** 직원 각자 ChatGPT에 **계약서** 붙여넣기  
**After:** **승인된** 사내 챗 + **문서만** 연결 + **접근 권한**  

## 오늘 해볼 것

업무에 AI 쓸 때 **한 줄 메모**:  
“이 내용 **외부 서비스**에 넣어도 되는 **공개 수준**인가?”  
애매하면 **넣지 않기** — 그게 사내 AI랑의 첫 차이예요.

## 카카오 요약

ChatGPT 잘 쓰는 것과 ‘회사 AI 도입’은 달라요. 개인 AI는 **내 공책**처럼 편한 대신, **회사 기밀·고객 정보**를 넣으면 위험해요. 사내 AI는 **도서관+출입증** — 어떤 문서를 쓰는지, 로그·규정이 있어요.

실무: 계약서·단가를 **개인 ChatGPT**에 붙이는 건 Before, **승인된 사내 도구 + 문서만 연결**이 After.

오늘부터 AI 쓸 때 **“이건 밖에 나가도 되는 정보?”** 한 번만 확인해 보세요.
