# 코드 없이 자동화, 어디까지?

## 첫 3초

“**자동화**”하면 **코딩**만 떠올리시나요?  
비개발자도 **여기까지**는 됩니다.

## 이런 적 있지 않나

매일 **같은 보고** 메일, **같은 시간**에 **리마인드**,  
**AI 요약**을 **카톡**으로 받고 싶은데 **손이 안 감**.

## 쉽게 말하면

자동화 = **“이 조건이면 이걸 해라”**를 **사람 대신** 돌리기.  

**코드 없이** 가능한 것:  
- **예약** (매일 7:30, 매주 월요일)  
- **연결** (GitHub → 카카오, 이메일 → 스프레드시트)  
- **AI** (저장된 글·템플릿 **전송**)  

**코드/개발** 쪽으로 갈수록: **조건이 복잡**, **사내 시스템** 깊게 연동.

## 회사에서 이렇게 쓰인다

**가능 (비개발):**  
- **매일 아침** 팀 채널에 **AI 한 줄 팁** (슬랙 예약 + 템플릿)  
- **구글 폼** 제출 → **담당자 메일** (노코드 도구)  
- **이 프로젝트**: GitHub **스켒줄** → **카톡 브리핑** (100% 무료)  

**개발 필요:**  
- **ERP** 재고 바뀔 때 **자동 발주**  
- **고객 데이터** 실시간 **AI 분류**  

**Before:** “**매일** AI 개념 **읽어야지**” → **잊음**  
**After:** **7:30 카톡** → **출근 지하철**에서 **1편**

## 오늘 해볼 것

**반복 3번 이상** 하는 일 하나 적고,  
“**사람 / 예약 / AI / 개발**” 중 **어디까지**면 되는지 **체크**해 보세요.

## 카카오 요약

자동화는 코딩만이 아니에요. **예약(매일 7:30)**, **도구 연결(카톡·메일)**, **저장된 AI 글 보내기**는 비개발자도 가능해요. ERP·실시간 데이터까지 가면 개발이 필요해요.

실무: 매일 팀 팁, 폼→메일 알림, **지금처럼 카톡 브리핑** = 노코드·저비용. “매일 읽어야지” 하다 잊는 Before → **7:30 카톡** After.

오늘: **세 번 이상 반복**하는 일 하나 골라, **예약만으로** 될지 적어 보세요.
